# GridClasses_Modding
This Repo is for people who want to mod my game.

Things found in this repo:

README: General Information

HowToMod: Full Guide on how to mod GridClasses
PlayerData: All the playerdata you are able to change
AllTileIDs: All the tile IDs that are standard in GridClasses

Everything is on the latest version.

Want to see how to mod earlier versions? Go to the GridClasses discord server and go to the channel #modding-info.
There will be all documents of modding my game from the earliest version!
